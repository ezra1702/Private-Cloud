Saya adalah mahasiswa Teknik Komputer yang sedang mengerjakan Tugas Akhir Mata Kuliah Administrasi Sistem Server.

Bantu saya merancang proyek secara lengkap dengan judul:

"Rancang Bangun Platform Enterprise Private Cloud Storage Berbasis Nextcloud dengan Integrasi Object Storage MinIO, Caching Redis, Keamanan SSL/TLS, dan High Availability Terotomatisasi Menggunakan Ansible Roles Berbasis Docker Container pada WSL2"

Proyek ini harus mengimplementasikan 5 topik mata kuliah berikut dengan cakupan industri:

1. Docker – Kontainer (Multi-Container Orchestration: Nextcloud, MariaDB, HAProxy, Redis, MinIO, Prometheus, Grafana)
2. Web Server (HAProxy/Nginx dengan HTTPS SSL/TLS Termination & Apache backend)
3. Manajemen User (Otentikasi internal Nextcloud, pembagian grup, hak akses, dan manajemen kuota penyimpanan)
4. Infrastructure as Code (Ansible Roles modular untuk otomatisasi deployment engine, config, dan container lifecycle)
5. High Availability (Round Robin Load Balancing, Session Stickiness, failover, stateless nodes via MinIO S3 API, dan Redis transactional locking)

Studi kasus:

Membangun platform private cloud storage yang berfungsi seperti Google Drive sederhana menggunakan Nextcloud. Sistem harus mendukung manajemen pengguna, upload dan download file melalui browser, deployment otomatis menggunakan Ansible, serta memiliki mekanisme high availability menggunakan beberapa container dan load balancer.

Lingkungan implementasi:

* Laptop pribadi
* WSL2 (Ubuntu 22.04 LTS) pada Windows Host
* Docker Engine dan Docker Compose Plugin
* Nextcloud (App Server)
* MariaDB (Relational Database)
* Redis (In-Memory Cache & File Locking)
* MinIO (S3-Compatible Object Storage)
* HAProxy atau Nginx (SSL/TLS Load Balancer)
* Prometheus & Grafana (Monitoring Stack)
* Ansible (Automated Deployment via Roles)

Arsitektur yang digunakan:

1 Load Balancer (HAProxy/Nginx dengan HTTPS)
2 Container Nextcloud (Replika Backend)
1 Database MariaDB (Relational Metadata DB)
1 Redis Container (In-Memory Caching & Locking)
1 MinIO Container (S3 Object Storage)
1 Prometheus Container & 1 Grafana Container (Monitoring)
1 Node Controller Ansible (WSL2 Localhost Connection)
